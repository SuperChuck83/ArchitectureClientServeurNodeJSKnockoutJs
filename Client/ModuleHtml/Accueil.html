Accueil KnockoutJs

Serveur Node js

Ce site est une architecture de projet node js, proposant une API Get/Post ainsi que la possibilité de connection avec des websocket.

L'IHM et les modules

Côté IHM c'est le framework knock out qui est utilisé, il permet de définir des modules d'ihm bien séparé du reste du code ( ici les modules d'inscription, connexion, de tchat, ou encore l'accueil du site )

Inscription / Connexion

Le module d'inscription permet d'apeller une URL,POST qui reçoit le pseudo et le mot de passe et enregistre l'utilisateur en base !
La connexion quant à elle permet de s'authentifier au prêt du socket, si on renseigne le bon pseudo et mdp la partie socket serveur conservera les données.

Tchat

Le module de tchat permet de voir une simple communication entre socket, on envois un message et tout le monde le reçoit, si on se connecte avec un pseudo le tchat affichera le pseudo sinon il affichera "Anonymous"
